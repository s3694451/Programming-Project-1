# PlayWith
Capstone Project at RMIT University, Semester 2 - 2020

Social Media web app where you match with people with the same interests as you

# Features:
- Profile Creation
- Login / Logout
- Post and Comment on your home page
- Join / Create Clans
- Realtime chat with Clans
- Security Features
- Admin Functionality
